5.2 使用JavaScript实现接口

使用JavaScript实现接口

应用场景

默认情况下SQLRestful接口使用SQL来实现，但某些场景下我们不需要使用SQL而是需要调用其他Restful微服务接口：

  • 多个Restful微服务接口的数据组合；
  • 需要对已有的Restful数据做脱敏等实现；
  • 第三方公有云服务接口的封装等；

配置方法

在接口定义上使用impl = "js"配置后可直接对exec或total实现采用JavaScript，如下所示：

  impl = "js"

  total = <<JS

  (function(){
    total = 0
    ...
    return total
  })()

  JS

  exec = <<JS
    var data
    ...
    return data
  JS

此处的JavaScript可以使用《使用JavaScript脚本》章节中说明的内置函数。
